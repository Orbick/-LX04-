# -LX04-
记录分享小米小爱触屏音箱LX04刷机过程以及安装软件，用Macro Deck控制电脑；
首先在闲鱼某商家购买了一台改装过的小米小爱触屏音箱（老款触屏音箱，新款的mini触屏音箱不能播放视频但是应该可以接入妙享中心），商家提供的改装选项有是否添加Type-C充电口（仅能充电不能传输数据），扩容（原4G内存扩成8G/16G/32G/64G/128G），最终选择C口+32G 165元的方案，并且商家以及完成刷机，安装有另外的安卓系统；
收到音箱后试用了商家预置的系统，感觉界面操作繁琐；
进入设置打开音箱的开发者模式，并将USB调试功能打开，允许USB传输数据；购入Micro转USB的数据线；安装搞机助手，根据搞机助手提示下载驱动，让软件能够识别设备；将音箱和电脑相连；
尝试安装软件：首先发现应用程序默认是通过Google Play安装的，并不能直接获取.apk安装包，于是乎找到 https://apkcombo.com/zh/ 网站能够下载.apk的文件；尝试通过搞机助手安装安装包，失败；通过搞机助手将安装包导入到音箱的文件目录中，在音箱中完成了软件安装；发现如果直接安装的话，只有.apk文件有效，.xapk安装不了，于是在音响中安装了类似于应用商城的应用程序APKPuere，实现了大部分软件的自由安装；
安装Macro Deck后，发现打开程序就是白屏加载不出来，起初怀疑是内存不够，但是查看发现运存占用只有十几兆（总运存1G）；接着怀疑是系统有问题，想重新更换新系统；
尝试自己刷新系统：在B站找到教程 【一分钱不花，小爱同学爆改STREAM DECK！】https://www.bilibili.com/video/BV1Yy411B7EF?vd_source=86e455f836877708486edd5cb14357c5 ，根据提示一步一步操作，其中有几点提出说明：一是路径不要出现中文；二是跟着操作将某文件放入根目录；三是刷入系统后，要断开数据线重新插一次让其启动（可能插上后要按+音量键），并等待几分钟（我遇到的很快，一两分钟就重启成功）；
继续尝试安装Macro Deck，发现还是进入程序后白屏加载不出来；接着在APKPure里面尝试安装低版本的Macro Deck，发现3.0.0以上的都情况一样，最早的Beta版本能进入程序，但是图标加载不出来；最终选择了一个高一点的Beta版本才大功告成！
